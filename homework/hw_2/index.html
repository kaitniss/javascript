Task 4

Напиши рядом с каждым выражением , тот ответ который по вашему мнению выведет console.
И потом сравните ваш результат с тем что на самом деле вывела консоль.

  • Infinity - "1"
  • "42" + 42
  • 2 + "1 1"
  • 99 + 101
  • "1" - "1"
  • "Result: " + 10/2
  • "3 + " bananas " + 2 + " apples "

Решение:

  1. Infinity - "1" => Infinity - максимально положительное число, почти при любых операциях, кроме деления 1 на infinity мы получаем его же.
  2. "42" + 42 => 4242 - строка, второе число приведется к строчному типу и случится конкотинация
  3. 2 + "1 1" => 21 1 - строка, первое число приведется к строчному типу и случится конкотинация в 21 пробел и 1 наследуютмся из переменной-строки
  4. 99 + 101 => оба типа numbers, складываем = 200
  5. "1" - "1" => строка и вычитание превращает переменные в numbers, = 0
  6. "Result: " + 10/2 =>Result: 5, сначала произойдет математическая операция, затем число 5 преобразуется в строку и приклеится к результату
  7. 3 + " bananas " + 2 + " apples " => все по порядку склеивается в строку, числа при сложении со строками преобразуются в строки. 3 bananas 2 apples
